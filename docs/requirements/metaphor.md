# Core Metaphor

ストレージを宇宙空間にマッピングする基本定義。

---

## Entity Mapping

| ストレージ | 宇宙 |
|-----------|------|
| 現在のフォルダ | 恒星（中心） |
| 子フォルダ | 惑星 |
| 子ファイル | 惑星（八面体） |
| 孫要素 | 衛星（リングで抽象化） |

---

## Property Mapping

| プロパティ | 表現 |
|-----------|------|
| サイズ | 天体の大きさ |
| 更新日時 | 明るさ（新しい = 明るい） |
| ファイル種別 | 色 |

---

## Operation Mapping

| エクスプローラー操作 | CLOSM Probe操作 |
|--------------------|----------------|
| フォルダを開く | 惑星をダブルクリック（ドリルダウン） |
| 親フォルダに戻る | 恒星をダブルクリック |
| ファイル情報を見る | 天体をクリック |
| スクロール | ズーム |
| ウィンドウをドラッグ | 軌道回転 |
